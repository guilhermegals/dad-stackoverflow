 O átomo é a menor parte de todo o sistema, como um botão, input, imagem e texto. 
 A molécula é a junção de alguns destes itens ou uma parte em que deva repetir esses átomos. 
    Aplicando em nosso contexto uma molécula poderia significar uma pergunta no feed, 
    Essa pergunta pode ser composta de imagem e texto (dois átomos).
 O organismo funcionaria como a junção de algumas moléculas. Como por exemplo o feed de posts, o header e etc.
 Por fim ele chama de template, mas no nosso projeto já existe um cara que cumpre o papel dele, os layouts.
